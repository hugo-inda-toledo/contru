 Luego de instalar vagrant, se debe correr el siguiente comando
 vagrant box add ubuntu_14.04 https://oss-binaries.phusionpassenger.com/vagrant/boxes/latest/ubuntu-14.04-amd64-vbox.box


*********************************Modificaciones del CSS******************************
En el proyecto se esta usando hojas de estilo LESS (http://lesscss.org/) para compilarlas en el archivo CSS final, para compilar tu propio CSS puedes instalar Koala (http://koala-app.com/).
